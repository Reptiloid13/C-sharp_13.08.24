Делегат - Это тип данных, который описывает сигнатуру метода

public delegate int Something(int a, int b);

Something Sum = (int a, int b) => 14; 
Something Sum = (int a, int b) => "14"; // Нельзя 

Событие - Вызов, оповещение, отправка сообщений всем заинтересованным. (Звонок) 
Событие объявляется внутри класса. Не внутри метода. 

Экземпляр класса — это объект, или его копия, которая находится в памяти компьютера.Программист создаёт экземпляр класса с помощью операции new.

static метод (относится и к полям и к свойствам) - тот метод, для вызова которого не нужно создавать экземпляр класса.

myStaticClass msc = new myStaticClass();  // НЕ сможем в Статик таким образом  вызывать экземпляр класса 
myStaticClass.someFunction(); // Правильное создание экземпляра класса

static классы - 1) Нельзя создавать экземпляр класса, используют ключевое слово new
2) Не разрешается использовать не статические члены этого же класса
3) Класс напечатан и не может быть унаследован от любого класса или интерфейса, кроме Object
4) Для доступа к члену статистического класса следует использовать имя класса, а не имя переменной


Конструктор - Конструктор инициализирует поля класса и  выполняется раньше чем код.
Свойства -  Специальный "ящик" Куда мы можем класть вещи (значения) с помощью set 
и доставать их с помощью get. Это помогает контролировать как тови данные используются и изменяются, делая код безопаснее и удобнее
Свойства решают проблему неправильного доступа к данным объекта, делая программу более безопасной

Виртуальный метод - это метод, который может быть переопределен в производных классах.
Имеет реализацию как в базовом классе, так и в производном классе.
Он используется, когда базовая функциональность метода одинакова,
но иногда в производном классе требуется больше функциональности. 

Полиморфизм — это возможность объекта изменять и дорабатывать методы родительского объекта с целью решения прикладной задачи.

Environment.CurrentDirectory -  Одним из способов получения текущей рабочей директории приложения является использование свойства Environment.CurrentDirectory.
Это свойство возвращает абсолютный путь к директории, в которой была запущена программа.



Инициализация - задать начальные значения 

foreach - foreach (Тип переменной in  коллекция)
		Тип переменной - это тип данных элементов в коллекции, которую перебираем
		Коллекция = это объект, который реализует интерфейс. Это может быть массив, список и тд. 

Пример - string[] fruits = { "яблоко", "банан", "апельсин" };

foreach (string fruit in fruits)
{
    Console.WriteLine(fruit);
}

Console.WriteLine () - То что пишется в круглых скобках называется аргумент или параметр
