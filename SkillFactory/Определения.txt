Делегат - Предоставляют такие объекты которые указывают на методы.
То есть делегаты - это указатели на методы с помощью делегатов мы можем вызывать данные методы. (Описание сигнатуры методов)

Событие - Вызов, оповещение, отправка сообщений всем заинтересованным. (Звонок) 
Событие объявляется внутри класса. Не внутри метода. 

Экземпляр класса — это объект, или его копия, которая находится в памяти компьютера.Программист создаёт экземпляр класса с помощью операции new.

static метод (относится и к полям и к свойствам) - тот метод, для вызова которого не нужно создавать экземпляр класса.

myStaticClass msc = new myStaticClass();  // НЕ сможем в Статик таким образом  вызывать экземпляр класса 
myStaticClass.someFunction(); // Правильное создание экземпляра класса

static классы - 1) Нельзя создавать экземпляр класса, используют ключевое слово new
2) Не разрешается использовать не статические члены этого же класса
3) Класс напечатан и не может быть унаследован от любого класса или интерфейса, кроме Object
4) Для доступа к члену статистического класса следует использовать имя класса, а не имя переменной
